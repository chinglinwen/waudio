# waudio
waudio convert English text to audio through ivona service

## Usage

./waudio -k "$ivona_access_key" -s "$ivona_secret_key" -f demo.txt

## Help
Usage of ./waudio:

	-f string
		data file
		
	-help
		show help
		
	-k string
		IVONA_ACCESS_KEY
		
	-s string
		IVONA_SECRET_KEY
		
	-v	version

## Notes
Suggest to register a ivona service access key and secret key to use it
